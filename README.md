Владимир Уточкин не представляет.# 👔 Style Advisor

**Персональный стилист в браузере** — приложение помогает выбрать одежду на сегодня на основе опроса и твоего гардероба.

## ✨ Возможности

- **Умный подбор лука** — ответь на 3 вопроса и получи готовый образ
- **Каталог гардероба** — добавляй свои вещи с фото, категориями и тегами
- **Локальное хранение** — данные только в твоём браузере, без серверов
- **Экспорт/Импорт** — скачай гардероб в JSON или загрузи на новом устройстве
- **Демо-образцы** — 10 предзагруженных вещей для первого теста
- **Адаптивный дизайн** — удобно на телефоне и на десктопе

## 🚀 Быстрый старт

Просто открой `index.html` в любом браузере, или [разверни на GitHub Pages](https://pages.github.com/).
